## Installation

```
npm i --save-dev react-string-checker
// OR
yarn add -D react-string-checker
```

## Starting

Insert into `package.json` file script:

```
...
"scripts": {
    ...
    "string-checker": "react-string-checker"
    ...
}
...
```

Run the command:

```
yarn string-checker
```
